  • Changelog

    • Lextend Changelog

      Version v4.8-0:
      • Improved discovery of the Sonos devices
      • Improved TuneIn interaction
      • If Radio station is saved under Radio Stations Learn Radio Stations now the Radio
        Station’s name is shown in the Lextend interface.
      • After the Radio station(s) is newly learnt in the Lextend, and Lextend has started the
        Radio Station, now the Image of the Radio station and the Radio stations name is
        shown in the Sonos App.
      • If the Radio Station is stopped for playing a doorbell, alarm etc. and then started
        again, now the Image of the Radio station and the Radio stations name is shown in
        the Sonos App.

      Version v4.8-1:
      • New icons on the mainpage
      • Added changelog

      Version v4.8-2:
      • Fixed custom mp3 file upload issue
      • Fixed rpc issue

      Version v4.8-3:
      • Fixed internal server error issue

      Version v4.8-4:
      • Fixed the issue with IP configuration

      Version v4.8-5:
      • Fixed the Alarm, LineIn and Doorbell handling preferences

      Version v4.8-6:
      • Fixed the UDP Control Device IP issue
      • Fixed the internal server issue with sonos Discover
      • Fixed the IP configuration issue with Raspbian jessie

      Version v4.8-7:
      • Solved the issue with don’t reload the track which have been played before Doorbell play when playing from the play list
      • Introduce a Reboot button under the General setting page which will reboot the system
      • Scheduled reboot section added under the general setting page and Introduce an enable button for the scheduled reboot and a section for set the reboot time.
      • Added a section for select the TIMEZONE and Display the time of lextend as per the selected TIMEZONE

      Version v4.8-8:
      • Fixed the issue of system reboot multiple times when the sheduled reboot time is set.

      Version v4.8-9:
      • Added Sonos Firmware Version 10.1 - Please make sure that all your Sonos run on Version 10.1. or higher

      Version v4.9-0:
      • Fixed the discovery of the Sonos devices

      Version v4.9-1:
      • OTA warning Message added for no internet connection

      Version v4.9-2:
      • Improved discovery of the Sonos devices

      Version v4.9-3:
      • Fixed the issue with sonos discovery

      Version v4.9-4:
      • Fixed the issue with factory reset

      Version v4.9-5:
      • Improved wireless connectivity

      Version v4.9-6:
      • Improved discovery of the Sonos devices

      Version v4.9-7:
      • Fixed the issue with sonos discovery

      Version v4.9-8:
      • Developed for Raspberry pi compute module

      Version v4.9-9:
      • Added sonos API
      • Added discover,play,pause,next track,previous track,load playlist and volume contol under sonos API
